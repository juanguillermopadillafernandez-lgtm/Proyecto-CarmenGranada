# Carmen | Red Intergeneracional Granada 2031 💃❤️

[![🌐 Ver Web en Vivo](https://img.shields.io/badge/🌐%20Ver%20Web%20en%20Vivo-Carmen%20Granada-A62639?style=for-the-badge)](https://juanguillermopadillafernandez-lgtm.github.io/Proyecto-CarmenGranada/)

![Granada — Alhambra y Sierra Nevada](https://images.unsplash.com/photo-1543783207-ec64e4d95325?auto=format&fit=crop&q=80&w=1600)

**Carmen** es una plataforma digital diseñada para conectar a jóvenes estudiantes con personas mayores en los barrios de Granada (comenzando por el Zaidín). El objetivo es combatir la brecha digital y la soledad no deseada, fomentando un intercambio de valor: **ayuda digital por puntos culturales**.

Este proyecto forma parte de la iniciativa para la **Capitalidad Europea de la Cultura Granada 2031**.

> 🔗 **Web en vivo:** [https://juanguillermopadillafernandez-lgtm.github.io/Proyecto-CarmenGranada/](https://juanguillermopadillafernandez-lgtm.github.io/Proyecto-CarmenGranada/)

---

## ✨ Características Principales

-   **🤝 Conexión Intergeneracional:** Une a dos generaciones con necesidades complementarias.
-   **📱 Alfabetización Digital:** Los jóvenes ayudan a los mayores con tareas como Bizum, citas médicas online, videollamadas y seguridad en la red.
-   **🎫 Sistema de Recompensas (Puntos Carmen):** Por cada hora de ayuda, los jóvenes reciben 50 puntos canjeables por entradas de cine, teatro, festivales y transporte.
-   **📜 Certificación de Competencias:** Los participantes reciben certificados oficiales validados por la Concejalía de Bienestar Social para enriquecer su CV.
-   **🛡️ Seguridad y Verificación:** Sistema de registro con verificación de identidad y formación previa obligatoria.
-   **📍 Enfoque Local:** Implementación por barrios, facilitando la cercanía y la creación de comunidad real.

## 🛠️ Tecnologías Utilizadas

El proyecto está construido con tecnologías web modernas de alto rendimiento y estética premium:

-   **HTML5 Semántico:** Estructura robusta y accesible.
-   **CSS3 (Vanilla):** Diseño personalizado con variables CSS, animaciones complejas y diseño responsive.
-   **JavaScript (ES6+):** Lógica interactiva sin dependencias externas pesadas.
    -   **Intersection Observer API:** Para efectos de "Scroll Reveal".
    -   **Custom Toasts:** Sistema de notificaciones personalizado.
    -   **Parallax & Counters:** Experiencia visual dinámica.
-   **Fuentes de Google Fonts:** *DM Serif Display* para elegancia y *Plus Jakarta Sans* para legibilidad.
-   **Font Awesome 6:** Iconografía moderna y consistente.

## 🚀 Cómo Ejecutar el Proyecto

Este es un proyecto estático. No requiere instalación de dependencias de Node.js.

1.  Clona el repositorio o descarga los archivos.
2.  Abre el archivo `index.html` en cualquier navegador moderno.
3.  *Opcional:* Si usas VS Code, te recomendamos la extensión **Live Server** para una mejor experiencia de desarrollo.
